# Maji-Ndogo_project
The Maji Ndogo Water Analysis project is a project designed to analyze water access, service coverage, and infrastructure conditions in the Maji Ndogo region. The objective of this project is to transform raw data into meaningful insights that support data-driven decision-making for water resource planning and service improvement.

This project demonstrates practical data analytics skills, including data preparation, data modeling, DAX calculations, and interactive dashboard design using Power BI.

<h4> Objectives </h4>

<ul>
  <li> Analyze water access and service levels across different regions </li>
<li>Identify underserved communities and infrastructure gaps</li>

<li>Visualize key performance indicators (KPIs) related to water availability</li>

<li>Demonstrate end-to-end Power BI data analysis and visualization skills</li>
